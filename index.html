Learn WEB DEVELOPMENT

WEB DEVELOPMENT begins with

HTML

HTML stands for Hyper Text Markup Language.

HTML provides basic structure for a web page.

HTML Example :
< !DOCTYPE html >
< html >
< head >
< title > HTML Page < /title >
< title >
< /head >
< body >
  
< h1 > This is heading. < /h1 >
< p > This is paragraph. < /p >
  
< /body >
< /html >

CSS

CSS stands for Cascading Style Sheet.

CSS is used to beautify the web page.

CSS Example :
body {
background-color : black ;
}
h1 {
color : white ;
text-align : center ;
}
p {
font-family : sans-serif ;
}

JavaScript

JavaScript is the actual programming language of web.

JavaScript is the most famous programming language.

JavaScript Example :
< button onclick = "myFunction()" > Click Here >
< /button >
< script >
function myFunction() {
let x = document.getElementById("demo");
x.style.fontSize = "20px"; x.style.color = "red";
}
< /script >
